# Portfolio-nextjs
Projet de gestion d'un portfolio en nextJS
## Stacks
* Next.js 16.2.7
## Commandes
* démarrage du container``docker compose up -d --build``
* connexion au bash : ``docker exec -it portfolio-nextjs bash``
* faire ``npm run dev``
## Usages
* interface sur ``http://localhost:3000/``
